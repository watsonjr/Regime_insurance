## Simulation code for:
# Adapting to Social-ecological Tipping Points with Insurance
### James R. Watson, Jack Buckner,... OMF crew
### Abstract:
Herein lies the code for the analysis of the window of opportunity for insurance to play a helping role in climate adaptation.
